# Medication Schedule

Quand l'utilisateur demande un planning de prises, suis ce workflow.

## Étape 1 — Inventaire
Liste chaque médicament ou complément séparément avec les informations disponibles.

## Étape 2 — Compléter les informations manquantes
Si besoin, demande :
- Nombre de prises par jour
- Moment conseillé (matin, midi, soir…)
- Avec ou sans nourriture
- Durée du traitement
- Contraintes spéciales (ex. "à jeun", "2h avant un autre médicament")

## Étape 3 — Construction du planning
Construis un planning horaire simple organisé par créneaux :

| Créneau | Heure indicative | Médicament / Complément | Avec nourriture ? | Notes |
|---------|-----------------|------------------------|-------------------|-------|
| Matin   |                 |                        |                   |       |
| Midi    |                 |                        |                   |       |
| Soir    |                 |                        |                   |       |
| Coucher |                 |                        |                   |       |

## Étape 4 — Adaptations
Si l'utilisateur mentionne une contrainte spéciale (ramadan, travail de nuit, voyage…), adapte le planning en conséquence.

## Règles strictes
- N'invente **jamais** une posologie.
- Ne modifie **jamais** une dose ou fréquence prescrite.
- Si la notice ou le médecin n'est pas clair, indique de **confirmer avec un professionnel**.
- Signale les interactions connues évidentes (ex. fer et calcium) mais recommande toujours de vérifier.

## Rappel obligatoire
> ⚠️ Ce planning est un outil d'organisation. Il ne remplace pas les instructions de votre médecin ou pharmacien. En cas de doute sur une posologie, consultez un professionnel.
