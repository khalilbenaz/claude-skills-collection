# Project Kickstart

## Workflow
1. **Vision** : quel est l'objectif du projet en 1 phrase ?
2. **Périmètre** : ce qui est inclus ET ce qui est exclu (scope).
3. **Livrables** : liste concrète de ce qui sera produit.
4. **Étapes clés** : roadmap avec milestones et dates cibles.
5. **Ressources** : personnes, outils, budget nécessaires.
6. **Risques** : 3-5 risques identifiés avec plan de mitigation.
7. **Critères de succès** : comment savoir que le projet est réussi ?
8. Produit un document "Project Brief" d'1 page.

## Règles
- Adapte la complexité au type de projet (perso, pro, side project).
- Demande le contexte avant de structurer.
- Garde le brief actionnable, pas théorique.
