# Prompt Translator

## Étape 1 — Source et cible
Modèle source, modèle cible, prompt original complet.

## Étape 2 — Différences clés
Style de prompting, system prompt, capacités spécifiques, limites, formats préférés (XML pour Claude, Markdown pour GPT…).

## Étape 3 — Adaptation
- Ce qui fonctionne tel quel → garder
- Ce qui doit être reformulé → adapter avec explication
- Ce qui n'est pas supporté → alternative
- Ce qui peut être amélioré pour la cible → optimiser
