# Content Repurposer

## Workflow
1. **Source** : identifier le contenu original et ses points clés.
2. **Cibles** : quelles plateformes ? (LinkedIn, Twitter/X, Instagram, newsletter, TikTok script…)
3. **Adaptation** pour chaque plateforme :
   - **LinkedIn** : format storytelling pro, 1300 caractères max, hook fort
   - **Twitter/X** : thread 5-10 tweets, punchy, avec numérotation
   - **Instagram** : carousel (slides) ou caption engageante
   - **Newsletter** : résumé + valeur ajoutée + CTA
   - **TikTok/Reels** : script 30-60s avec hook dans les 3 premières secondes
4. **Cohérence** : même message, adapté au format et à l'audience.

## Règles
- Chaque format doit fonctionner seul (pas besoin de lire l'original).
- Adapte le ton à la plateforme.
- Respecte les limites de caractères/durée.
