Installation

Pret en 30 secondes.

Clone, package, importe. C'est tout.

01

Cloner

Clone le repo sur ta machine pour acceder aux 296 skills.

02

Packager

Zip le SKILL.md du skill voulu en fichier .skill.

03

Importer

Importe le .skill dans Claude via Parametres → Skills.

# Cloner la collection
git clone https://github.com/khalilbenaz/claude-skills-collection.git

# Packager un skill (exemple : gRPC)
cd claude-skills-collection/dev-skills/grpc-service-designer
zip ../../grpc-service-designer.skill SKILL.md

# Importer dans Claude.ai → Parametres → Skills
