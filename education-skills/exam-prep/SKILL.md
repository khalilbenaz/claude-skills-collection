# Exam Prep

## Workflow
1. **Sujet** : quel examen, quelle matière, quel chapitre ?
2. **Points clés** : résumé des concepts essentiels à maîtriser.
3. **Quiz** : 10-20 questions variées :
   - QCM (4 options)
   - Questions ouvertes courtes
   - Vrai/Faux avec justification
   - Exercices d'application
4. **Correction détaillée** : réponse + explication pour chaque question.
5. **Score et analyse** : points forts, points faibles, quoi retravailler.
6. **Plan de rattrapage** : cibler les lacunes identifiées.

## Règles
- Varie la difficulté (facile → moyen → difficile).
- Questions pièges uniquement si l'examen en contient.
- Adapte au format réel de l'examen si connu.
